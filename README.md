# Weather Diary — Дневник погоды

Автор: Пашпекин Константин Константинович

## Описание

Графическое приложение для ведения дневника погоды. Позволяет добавлять записи о погоде, просматривать их в таблице, фильтровать по дате и температуре, а также сохранять данные в файл JSON.

## Как запустить

Требуется Python 3.10 или новее (tkinter входит в стандартную библиотеку).

```
python app.py
```

```
weather_diary/

 README.md
```

## Примеры использования

**Добавление записи:**
1. Ввести дату в формате ДД.ММ.ГГГГ, например `15.06.2025`
2. Ввести температуру, например `-3` или `24.5`
3. Ввести описание, например `Облачно, ветер`
4. Выбрать осадки: Да / Нет
5. Нажать «Добавить запись»

**Фильтрация по дате:**
Ввести дату в поле «По дате», нажать «Применить» — покажет все записи за этот день.

**Фильтрация по температуре:**
Ввести число в поле «Температура выше», нажать «Применить» — покажет записи, где температура выше указанного значения.

**Сброс фильтра:**
Нажать «Сброс» — вернётся полный список.

## Формат данных (weather_data.json)

```json
[
  {
    "date": "15.06.2025",
    "temp": 24.0,
    "desc": "Ясно, тепло",
    "precip": "Нет"
  }
]
```

## Валидация

- Дата — обязательна, формат ДД.ММ.ГГГГ, должна быть реальной датой
- Температура — обязательна, должна быть числом
- Описание — не может быть пустым
